# Selenium E-commerce Automation Project

 This project demonstrates automation testing of an e-commerence application using Selenium WebDriver with java.
 
 Features
 Login functionality(valid & invalid)
 Add product to cart
 Cart validation
 Logout functionality

 Technologies Used
 Java 
 Selenium WebDriver
 Maven
 IntelliJ IDEA

 Project Structure
 src-->main/java/pages
    -->test/java

  How to Run
  Open project in IntelliJ IDEA
  Run test class

  ##Manual Test Cases
  This project also includes manual testing for the  e-commerce application
  ##Test Scenarios Covered
  Valid Login
  Invalid Login
  Add product to Cart
  Cart Validation
  Logout Functionality
